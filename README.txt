HOW TO USE?
- Pastikan folder digilib (yang sudah di ekstrak) berada di dalam folder htdocs
- Pastikan anda sudah memiliki database digilib. (Bila belum, silakan buat database dengan nama "digilib", lalu import "digilib.sql"
- Jalankan Apache & Mysql melalui xampp
- Akses Localhost/digilib
